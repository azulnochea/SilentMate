SilentMate

Time-Based Auto Silent Mode App

\ud83d\udcf1 프로젝트 소개

SilentMate는 사용자의 시간표에 따라 자동으로 휴대폰을 무음 모드로 전환해주는 안드로이드 앱입니다.

\ud83c\udf1f 주요 기능

  • 수업 시간 기반 자동 무음 전환
  • 요일/시간 수동 설정 가능
  • 백그라운드 서비스 실행
  • 알림 설정 및 간단한 UI

\ud83d\udee0\ufe0f 기술 스택

  • Frontend: Android Studio (XML)
  • Backend: Kotlin
  • Version Control: Git + GitHub
